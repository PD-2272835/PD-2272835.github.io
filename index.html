My Projects

Shrimp Games Framework

My custom rendering and physics framework that I built for MEGA and PROGC, SGF uses SDL3 as a portability layer to run on both PC and PS5! All mathematics are powered by Shrimp Games Math

Shrimp Games Math

My custom C++17 Maths library built for MEGA. Used extensively for rendering, entity management and collision detection in SGF. Works with both GNU and LLVM-based compilers for the C++17 standard

PRAPI

My first steps into graphics programming! An OpenGL rendering framework, used to create an orbit sim. Uses GLM for mathematics and features a scene graph for dirty-flag optimisation

Let Them Eat Cake!

A little game that myself and a group of friends made for the University of Portsmouth Game Jam 2025! Throw cake-based projectiles at the revolting peasants from your high castle. I built the cake selection, menu UI and damage system
